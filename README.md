# CRUD WITH LAMBDA FUNCTIONS AND MONGODB- AWS 

## Requirements

[Docker](https://www.docker.com/)  
[Python 3.7](https://www.python.org/)  
[NodeJS](https://nodejs.org/es/)  
[ServerlessFramework](https://serverless.com/)  

## Installation 

```bash
pip install -r requirements.txt
```

```bash
npm install 
```

## Deploy

```bash
serverless deploy 
```
